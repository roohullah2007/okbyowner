# Images Directory

This directory contains all the images used in the OK BY OWNER application.

## Required Images

### Logo
- **File**: `logo.png`
- **Dimensions**: 189px × 45px
- **Description**: Main logo for the header

### Hero Section
- **File**: `bg-image.webp`
- **Description**: Background image for the homepage hero section
- **Recommended Size**: 1920px × 800px or larger
- **Format**: WebP, JPG or PNG

### Selling Section
- **File**: `for-sale.webp`
- **Description**: Image showing "For Sale" sign for the selling section
- **Recommended Size**: 600px × 400px or larger
- **Format**: WebP, JPG or PNG

### Properties Section
- **Files**: `property-1.jpg`, `property-2.jpg`, `property-3.jpg`, `property-4.jpg`, `property-5.jpg`, `property-6.jpg`
- **Description**: Property listing images for the featured properties carousel
- **Recommended Size**: 800px × 600px or larger
- **Format**: JPG, PNG or WebP
- **Note**: Add more property images as needed following the naming convention `property-X.jpg`

## How to Add Images

1. Place your images in this directory (`public/images/`)
2. Reference them in your components using `/images/filename.ext`
3. Example: `<img src="/images/logo.png" alt="Logo" />`

## Notes

- Images in the `public` directory are publicly accessible
- Use web-optimized formats (WebP, optimized JPG/PNG)
- Keep file sizes reasonable for web performance
